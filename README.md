## 🛠️ Requisitos
- Ambos dispositivos deben estar conectados a la **misma red Wi-Fi**, no es magia.
- En PC: Windows 10 o superior y 470mb disponibles (por ahora...).
- En Celular: Android 8.0 o superior y 45mb disponibles (por ahora...).

## 👨‍💻 Sobre el Proyecto
Desarrollado por **Valentín Villanueva** para simplificar el envío de archivos localmente, y depaso aprender nuevas tecnologias incluyendo **Electron, Android Studio y Capacitor**, la app esta construida totalmente en JS/HTML/CSS.
